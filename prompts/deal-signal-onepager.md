# One-page deal signal (on-demand)

When the user names a company, product, or domain:

## Sections

1. **Overview**: what they build, category, stage (if public)
2. **Differentiation**: vs 2–3 named alternatives (public knowledge only)
3. **Traction signals** (only if sourced): users, revenue claims, partnerships, benchmarks
4. **Team** (public bios only; no doxxing)
5. **Risks**: tech, market, regulation, geopolitics, cap table unknowns
6. **Suggested diligence**: experts to speak with, metrics to request, 3–5 questions
7. **Sources**: bullet list of URLs

## Rules

- Not investment advice; no insider or non-public data.
- If data is missing, write **Unknown** — do not infer private terms.
