# 将单条（或同一作者的少量）X 推文改写为中文摘要块

输入：`feed-investor.json` 里 `type: social_en` 条目，通常 `title` 为推文全文，`url` 指向 `x.com`，可选 `authorHandle`、`authorName`、`tags`（`ai_builder` / `ai_investor`）。

## 输出格式（严格按顺序）

1. **抬头一行**：`**{authorName 或 display}**（@{authorHandle}）`  
2. **综述**：1–3 句中文，概括帖子主题与语境；若推文含链接/线程，说明「指向某产品/论文/讨论」即可，勿臆测未给出的正文。  
3. **一句话结论**：单独起行，格式为 `**一句话结论：** …` — 用一句话说清「对读者意味着什么」或「核心论断」。  
4. **关键要点**：  
   ```
   关键要点：
   - …
   - …
   ```  
   共 **3–5** 条 bullet；短句；可含可验证事实与作者观点（标注为观点）。  
5. **链接**：最后一行或多行，使用条目中的完整 `url`。

## 规则

- **禁止**大段照抄英文推文；允许保留必要专有名、产品名、论文名（可附中文括注）。  
- **无** `authorHandle` 时从 `source`（如 `twitter_karpathy`）推断 `@karpathy`。  
- 信息不足时少写 bullet，不要编造数字、融资、未出现的机构。  
- 若 `tags` 含 `ai_investor`，可点一句「机构叙事 / 赛道观点」类提醒（仍为摘要，非投资建议）。
